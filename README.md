# Deprem Tahmin Uygulaması / Earthquake Prediction Application 


## Getting Started

Flutter Dili ile UI tasarlanmıştır. / UI is designed with Flutter.
###
Python ile Facebook Prophet Kütüphanesi kullanılarak deprem tahmini yapılmıştır. / Earthquake predictions were made using Python and Facebook's Prophet Library. 
###
İlk açışta "pubspec.yaml" dosyasını get yapmayı unutmayın. / Don't forget to get the "pubspec.yaml" file.
